
# Местоимения

Местоимения - это слова, которые замещают содержательные слова (или фразы из содержательных слов).

## Личные местоимения

| Singular    | Plural       |
|:------------|:-------------|
| **mi**      | **vi**       |
| я           | мы           |
| **tu**      | **yu**       |
| ты          | вы           |
| **da**      | **di**       |
| он, она, оно| они          |
